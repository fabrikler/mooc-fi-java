# Java Programming MOOC – University of Helsinki

Solutions and exercises from the Java Programming MOOC by the University of Helsinki.
Completed as part of my FIAE (Software Development) apprenticeship in Berlin.

## What I'm learning

- Object-oriented programming fundamentals: classes, inheritance, interfaces
- Core data structures and problem-solving in a strongly typed language
- Writing clean, structured, and maintainable code through practical exercises

These concepts are directly relevant for my apprenticeship exams and form the foundation for deeper work in algorithms, backend systems, and infrastructure.
